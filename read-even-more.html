even more information

Try ensemble now by installing our test application. The README text describes the steps you need to take to install ensemble. You might also look into our kernel and admin modules since they provide the core of ensemble. A complete list of ensemble modules can you find at our wiki.

contribute

If you want to fix a bug or add features to ensemble, please do so! Feel free to fork ensemble and modify it to your needs. It is advised to not touch ensemble's code when building applications, but overwrite the code with your own modules instead. More about that can be found at our wiki.

However, bugs need to be fixed and features need to be developed. To do so, create a branch in your clone and give it a descriptive name. For bugfixes you might want to use hotfix/some-description and for features feature/some-description. If you finished your work, feel free to open a Pull Request and let us review your work and merge it back into our repository. More information about this workflow can be found at out wiki.

intiators
